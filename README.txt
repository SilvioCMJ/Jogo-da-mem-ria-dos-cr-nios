
Jogo dos Crânios

Descrição
Jogo dos Crânios é um jogo da memória didático voltado para o aprendizado de biologia. O jogo consiste em parear cartas: uma com a imagem do crânio de um animal e a outra com a imagem do próprio animal. Com este projeto, os jogadores podem aprender sobre diferentes espécies de forma divertida e interativa.

Funcionalidades
- Jogo da memória com 15 pares de cartas.
- Sistema de pontuação para dois jogadores.
- Interface simples e intuitiva.
- Jogo responsivo utilizando HTML, CSS e JavaScript.

Como Jogar
1. Abra o arquivo index.html em um navegador.
2. O jogo começa com todas as cartas viradas para baixo.
3. Dois jogadores se alternam tentando encontrar pares de cartas (um crânio e seu animal correspondente).
4. Se os jogadores acertarem o par, ganham pontos e jogam novamente. Caso contrário, o próximo jogador tenta.
5. O jogo termina quando todas as cartas forem encontradas. O jogador com mais pontos vence!

Tecnologias Utilizadas
- HTML: Estrutura do jogo.
- CSS: Estilização do layout e design responsivo.
- JavaScript: Lógica do jogo e controle da pontuação.

Como Executar o Projeto
1. Clone o repositório:
   git clone https://github.com/seu-usuario/jogo-dos-cranios.git
2. Navegue até o diretório do projeto:
   cd jogo-dos-cranios
3. Abra o arquivo index.html em seu navegador.
